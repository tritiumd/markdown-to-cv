# Write CV on Markdown

## Tech stack
- Python 3.11
- Flask 3.0.3
- HTML, CSS

## Features
- Drag and drop to upload files
- Download CV in PDF format

## How to run
1. Clone this repository
2. Install dependencies
    ```bash
    python -m venv venv
    source venv/bin/activate
    pip install -r requirements.txt
    ``` 
3. Run the backend
    ```bash
   flask run
    ```
